# Hand Tracking Experience

## 🚀 Overview
Interactive hand tracking project with neon air drawing, glowing particles, and elastic strings.

## ✨ Features
- Hand tracking using MediaPipe
- Neon air drawing
- Particle effects
- Elastic string simulation

## 🛠 Tech Stack
- JavaScript
- Three.js / Canvas
- MediaPipe

## ▶️ How to Run
1. Open index.html
2. Allow camera access
3. Start interacting
